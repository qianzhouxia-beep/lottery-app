# 彩票预测系统 - 产品设计规格书
**版本: v2.0 · 2026-06-10**

---

## 1. 设计理念

### 产品定位
> **你的私人彩票分析助理** — 不是玄学工具，而是数据驱动的智能分析平台。

### 设计原则
1. **信息优先** — 数据可视化 > 装饰性 UI
2. **极简上手** — 用户无需理解算法，打开就用
3. **信任感** — 透明展示数据来源、模型逻辑、预测置信度
4. **移动优先** — 单手操作，拇指友好

### 情感定位
专业、可信赖、略带科技感。不像彩票站，而像 Bloomberg Terminal + Apple Weather 的结合。

## 2. 设计语言

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 (大乐透) | `#6366F1` (Indigo) | 科技感、专业 |
| 主色 (双色球) | `#EF4444` (Red) | 热度、活力 |
| 深色背景 | `#0F172A` | 主要背景色 |
| 卡片背景 | `#1E293B` | 深色模式卡片 |
| 浅色模式 | `#F8FAFC` / `#FFFFFF` | 可选主题 |
| 成功/热号 | `#10B981` (Emerald) | 正选/热号 |
| 警告/冷号 | `#F59E0B` (Amber) | 反选/冷号 |
| 文字主色 | `#F1F5F9` (深色) / `#1E293B` (浅色) | |
| 文字次色 | `#94A3B8` | 辅助信息 |

### 字体
- **主字体**: `"Inter", -apple-system, "PingFang SC", sans-serif`
- **数字字体**: `"JetBrains Mono", "SF Mono", monospace` — 号码用等宽字体，更有数据感
- **字号层级**:
  - 标题: 1.25rem / 700
  - 副标题: 1rem / 600
  - 正文: 0.9rem / 400
  - 辅助: 0.75rem / 400

### 间距系统
- 基础单位: 4px
- 卡片内边距: 16px
- 卡片间距: 12px
- 区块间距: 24px
- 圆角: 16px (卡片) / 50% (球) / 8px (按钮)

### 动效哲学
- **意图明确**: 每个动画都有功能目的（状态切换 / 加载反馈 / 过渡引导）
- **快速响应**: 150ms ~ 250ms，避免拖沓
- **入场动画**: 卡片从下方滑入 (translateY 20px → 0, 200ms ease-out, staggered 50ms)
- **数字变化**: 数字切换时短暂缩放 (scale 1.1 → 1, 150ms)
- **加载态**: 骨架屏 + 脉冲动画，不用转圈

## 3. 页面结构

### 导航架构 (底部 Tab 栏)

```
┌─────────────────────────────────────────────┐
│  [状态栏: 服务状态 + 最后更新时间]            │
├─────────────────────────────────────────────┤
│                                             │
│              主内容区域                      │
│           (可滚动，下拉刷新)                 │
│                                             │
├─────────────────────────────────────────────┤
│  [首页] [预测] [历史] [设置]                 │
└─────────────────────────────────────────────┘
```

**4个 Tab:**
1. **首页** — 最新开奖 + 趋势摘要 + 快速预测入口
2. **预测** — 完整预测生成器，含正选/反选/历史对比
3. **历史** — 开奖历史 + 复盘工具
4. **设置** — 彩种切换 / 主题 / 数据更新 / 关于

## 5. 组件库

### 5.1 球组件 (Ball)

**状态:**
- 前区红球: 渐变红 (#EF4444 → #DC2626)，白字
- 后区蓝球: 渐变蓝 (#3B82F6 → #2563EB)，白字
- 选中态: 外发光 (box-shadow: 0 0 0 3px var(--primary))
- 禁用态: 50% 透明度

**尺寸:**
- 大号: 48px (首页大卡片)
- 中号: 40px (预测页)
- 小号: 32px (历史表格 / 热号列表)

### 5.2 卡片组件 (Card)

- 背景: 深色模式 `#1E293B`，浅色模式 `#FFFFFF`
- 边框: 1px solid `#334155` (深色) / `#E2E8F0` (浅色)
- 圆角: 16px
- 内边距: 16px
- 阴影: `0 4px 6px -1px rgba(0,0,0,0.1)`

### 5.3 按钮组件 (Button)

| 类型 | 样式 |
|------|------|
| 主按钮 | 渐变背景 (主色)，白字，悬停上移 + 阴影 |
| 次按钮 | 透明背景 + 边框，悬停填充 |
| 图标按钮 | 圆形，40px，浅色背景 |
| 危险按钮 | 红色背景 (用于清空等操作) |

### 5.4 进度条 (Confidence Bar)

- 背景: `#334155`
- 填充: 渐变色 (绿 → 黄 → 红，基于置信度)
- 高度: 8px，圆角 4px
- 动画: 宽度从 0 渐变到目标值，300ms ease-out

### 5.5 骨架屏 (Skeleton)

- 背景: 渐变动画 (`#1E293B` → `#334155` → `#1E293B`)，1.5s 循环
- 形状: 与实际内容相同的圆角矩形
- 用于: 加载中状态的数据卡

## 7. 技术规格

### 前端
- 纯 HTML + CSS + JavaScript（无框架依赖）
- 单文件架构 (`index.html`)，便于部署
- CSS 变量支持主题切换
- 响应式: 优先移动端 (max-width: 480px)

### 后端 API
- 端口: 5123
- 协议: HTTP/1.1 (raw socket 实现)
- 响应格式: JSON
- CORS: 支持跨域

### API 端点
| 端点 | 说明 | 响应 |
|------|------|------|
| `GET /` | 前端页面 | HTML |
| `GET /api/status` | 服务状态 | `{dlt_count, ssq_count, ...}` |
| `GET /api/latest?kind=dlt\|ssq` | 最新开奖 | `{period, front[], back[], sum, ...}` |
| `GET /api/history?kind=dlt&limit=30` | 历史记录 | `[{period, front[], back[], sum}, ...]` |
| `GET /api/trend?kind=dlt` | 趋势分析 | `{zone_dist, hot_front[], hot_back[], ...}` |
| `GET /api/predict?kind=dlt` | 生成预测 | `{for_period, predictions[], model, ...}` |
| `GET /api/review?kind=dlt&period=26063` | 复盘指定期 | `{actual, prev, ...}` |

### 数据文件
- `data/dlt_history.json` — 大乐透历史
- `data/ssq_history.json` — 双色球历史
- 字段: `period`, `front[]`, `back[]`, `sum`

## 8. 实现优先级

### Phase 1 (MVP)
1. 深色主题完整重构 (index.html)
2. Raw socket API 服务器稳定运行
3. 4个 Tab 页面完整功能
4. 球组件 + 卡片组件

### Phase 2 (完善)
1. 趋势走势图 (Canvas 或 SVG)
2. 置信度进度条
3. 复制到剪贴板功能
4. 历史复盘记录

### Phase 3 (高级)
1. 主题切换 (深色 / 浅色)
2. 预测历史本地存储 (localStorage)
3. 下拉刷新
4. PWA 支持 (离线缓存)
